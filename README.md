# SPR_ROLLO --------> POWODZENIA :D

1. Dołącz do zdalnego repozytorium SPR_ROLLO 
2. Sklonuj repozytorium SPR_ROLLO do swojego lokalnego katalogu roboczego.
3. Utwórz nową gałąź (branch) o nazwie swojego nazwiska.
4. Przełącz się na utworzoną gałąź.
5. Otwórz wskazany plik tekstowy .txt znajdujący się w repozytorium. Dodaj do pliku nowy wpis zawierający:
	swój numer z dziennika,
	po spacji imię i nazwisko.
Umieść swój wpis w odpowiednim miejscu w pliku, tak aby numery w pliku pozostały posortowane rosnąco.
Zapisz zmiany w pliku.
6. Wykonaj commit zmian z czytelnym komentarzem opisującym wprowadzone modyfikacje.
7. Wykonaj push zmian wyłącznie na swój branch (nazwany nazwiskiem).
8. Po poprawnym wysłaniu zmian utwórz Merge Request / Pull Request do gałęzi main:
operację merge wykonuj wyłącznie przez przeglądarkę,
w przypadku konfliktów rozwiąż je samodzielnie, zachowując dane wszystkich uczniów.
Zatwierdź merge do gałęzi main.

UWAGA!!!!!!!!!!!
Zabronione jest usuwanie lub modyfikowanie wpisów innych osób. 
W przypadku usunięcia pracy innych osób -> Automatycznie "bańkę łap xD"
